My Computer

CPU - Memory - Disk

Think of these concepts as an encycolpedia library.
In using this library you have physical books that contain data.
These physical books can be viewed as the DISK.
In order to access the DISK you must physically take them from the shelf, open to a certain page, and locate a topic.
This process can be considered MEMORY.
The MEMORY can be thought of as a bookmarker; a temporary, fluid hold on a page or topic.
In order to make use of the bookmarked data it must be processed.
In this example we use our eyes and brain as the CPU to read and comprehend the data.
